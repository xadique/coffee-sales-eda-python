# 📌 Project Overview

This project presents a comprehensive **Exploratory Data Analysis (EDA)** of a Coffee Sales dataset using **Python**.

The objective of this project is to transform raw sales data into meaningful business insights by performing data cleaning, preprocessing, statistical analysis, and visualization.

The analysis helps understand customer purchasing behavior, product demand, sales performance, payment preferences, and overall business trends.

---

# 🎯 Objectives

- Understand the dataset structure
- Perform data cleaning and preprocessing
- Handle missing values
- Replace invalid values (UNKNOWN & ERROR)
- Convert incorrect data types
- Perform Exploratory Data Analysis (EDA)
- Identify customer purchasing patterns
- Analyze payment methods
- Compare sales across locations
- Generate business insights
- Provide business recommendations

---

# 📌 Business Insights

Some important insights obtained from the analysis include:

- Customers mostly purchase small quantities.
- Quantity has a strong positive relationship with Total Spent.
- Digital payment methods are becoming increasingly popular.
- Some store locations generate significantly higher revenue.
- Consistent pricing strategy improves customer purchasing behavior.
- Inventory should prioritize frequently purchased products.

---

# 💼 Business Recommendations

- Maintain inventory for top-selling products.
- Promote high-demand items.
- Improve low-performing store locations.
- Encourage digital payment methods.
- Develop interactive dashboards.
- Implement predictive sales forecasting.
